# rest-api-client-app

Restore npm_module
npm install

Create React App Command
npx create-react-app rest-api-client

Change Directory
cd .\rest-api-client\

Install Axios
npm install axios

Install Bootstrap
npm install bootstrap

To Run React APP
npm start

Install Nuget Package at Backend API
Microsoft.AspNetCore.Cors
